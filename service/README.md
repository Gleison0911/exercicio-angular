# Serviço

Pequeno serviço para ser consumido pelo front em angular

## Como executar

```bash
npm install
npm run dev
```

## Como testar

```bash
npm install
npm run test:coverage
```

## Melhorias da camada de serviço

- Evoluir o banco de dados utilizando as migrações do knex
- Implementar autenticação com JWT
